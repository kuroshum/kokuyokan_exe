# Kokuyokan_exe
黒洋館というゲームのリポジトリです。  
青鬼、バイオハザード1を参考に制作した2D脱出アクションゲームです。  
このリポジトリには、実行ファイルを置いていますが、https://github.com/kuroshum/Kokuyokan_source でソースコードも公開しています。

# Dependecy  
GameEngine : Unity  
使用言語    : C#  
  
# Usage
デュアルショック4での操作  
左スティック : 移動
十字キー左右 : 武器選択
〇ボタン     : 攻撃
△ボタン     : 調べる
□ Xボタン    : 回復
